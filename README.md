# Curso TypeScript

## Executar arquivo

* tsc arquivo.ts
    * tsc -w (monitorar alterações no arquivo)
* node arquivo.js
    * criar script "start" para dependência live-server

## Executar com code runner (extensão)

* instalar `npm i -g ts-node`
* executar arquivo com `ctrl + alt + n`
